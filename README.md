# ExtinctionMarine 🦖 – High-Performance 2D Isometric Bullet Hell Simulation

A high-performance, architecture-driven 2D isometric bullet hell simulation (inspired by *Vampire Survivors*) built with **C#** and **Unity 6 LTS**. This project is designed with a strict emphasis on enterprise-grade software engineering principles, memory management optimization, and clean domain separation.

---


## 🚀 Tech Stack

* **Language:** C# (.NET 8+ compliant compiler features)
* **IDE:** Visual Studio 2026 Insiders
* **Engine Framework:** Unity 6 LTS (Universal Render Pipeline 2D)
* **Input Architecture:** Unity Input System Package (Event-Driven Architecture)
* **Data Persistence:** Asynchronous JSON Serialization (`System.IO` Streams)
